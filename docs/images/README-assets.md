## Adastra Systems – Image Assets

Place your brand assets in this folder using the exact filenames below so the site picks them up automatically:

- `stars-background.jpg` – dark, starry space background used for the full-page backdrop.
- `logo-adastra.svg` – Adastra Systems logo (ideally SVG with transparent background).
- `founder-benjamin.jpg` – Founder headshot for the “Meet the Founder” section.

You can change these filenames if you prefer, but then you must also update the corresponding `src` paths in:

- `app/components/BackgroundCanvas.js` – background image path
- `app/components/Logo.js` – logo image path
- `app/page.js` – founder image path
