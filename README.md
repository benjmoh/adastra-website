## Adastra Systems – AI Automation Landing Page

One-page, premium, dark-space themed marketing site for **Adastra Systems**, an AI automation consultancy.

### Tech Stack
- Next.js (App Router)
- React
- Tailwind CSS

### Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.

### Customising Brand Assets

- **Background image**: Replace the file at `public/images/stars-background.jpg` with your own star-field image, keeping the same filename.
- **Logo**: Replace `public/images/logo-adastra.svg` with your logo (or update the `Logo` component in `app/components/Logo.tsx` to point to a different file).
- **Founder headshot**: Replace `public/images/founder-benjamin.jpg` with your photo (or update the `FounderSection` component to use a different path).

All key brand assets are clearly commented in the code so you can swap them easily.
